# Python

## 课程大纲

本课程将会介绍一些基本的编程知识，所涉及的语言主要是python。通过学习本课程可以掌握基本的编程命令、基本的算法、以及基本的git命令。本课程主要知识点如下。

- 什么是编程，和计算机有什么关系
- python可以做什么
- 如何安装python并配置环境变量
- python脚本和命令行
- python基础语法
- python变量类型
- python运算符

## 课件

每节课的课件都会在课后上传到github上，请及时通过```git fetch```下载到本地（课件在"materials")
